Academic Planning Made Simple

TermTacklr is a .NET MAUI mobile application designed to help students manage academic terms, courses, and assessments in one organized location.

```
```

Key Features

Manage Terms

Create, edit, delete, and search academic terms.

Manage Courses

Track course details, instructor information, status, and notes.

Manage Assessments

Add and track objective and performance assessments.

Notifications

Receive reminders for upcoming course and assessment dates.

Reports

Generate reports to track academic progress.

Share Notes

Share course notes using built-in mobile sharing features.

Download the Android APK

Download the signed Android APK file below to install TermTacklr on an Android device.

Download TermTacklr APK

Deployment Notes

This application is deployed using GitHub Pages as a static hosting site for the Android APK. Containers were not necessary because TermTacklr is a mobile application that uses local SQLite storage and does not require a hosted backend service.
