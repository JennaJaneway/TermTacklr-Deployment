TermTacklr

WGU D424 Software Engineering Capstone Mobile Application

About the Application

TermTacklr is a .NET MAUI mobile application designed to help students manage academic terms, courses, and assessments in one organized location.

Key Features

  • Create, edit, delete, and search terms, courses, and assessments
  • Local SQLite database storage
  • Course and assessment notifications
  • Reports for academic tracking
  • Share Notes functionality
  • Validation, security features, and scalable design elements

Deployment Notes

This application is deployed using GitHub Pages as a static hosting site for the Android APK. Containers were not necessary because TermTacklr is a mobile application that uses local SQLite storage and does not require a hosted backend service.
