TermTacklr

```

TermTacklr is a .NET MAUI mobile application created for the WGU D424 Software Engineering Capstone. The app helps users manage academic terms, courses, and assessments.

Download

Download TermTacklr Android APK

Deployment Notes

This application is deployed using GitHub Pages as a static hosting site for the Android APK. Containers were not necessary because this is a mobile application with local SQLite storage and no hosted backend service.
```
